# New Project Skill

一键启动新项目，完成从需求澄清到开发启动的完整流程。

## 触发方式

```bash
/skill:new-project
```

或关键词：
- "开始新项目"
- "从零开始"
- "启动新项目"

## 执行流程

```
┌─────────────────────────────────────────────────────────────┐
│  Step 1: cost-control                                       │
│  └─ 建立成本基线                                             │
├─────────────────────────────────────────────────────────────┤
│  Step 2: socratic-inquiry                                   │
│  ├─ 苏格拉底式审问需求                                       │
│  ├─ 澄清项目目标、范围、约束                                  │
│  └─ 输出需求确认清单                                         │
├─────────────────────────────────────────────────────────────┤
│  Step 3: task-planner                                       │
│  ├─ 制定项目架构计划                                         │
│  ├─ 技术选型建议                                             │
│  ├─ 保存计划到 plan.md                                      │
│  └─ 等待用户确认                                             │
├─────────────────────────────────────────────────────────────┤
│  Step 4: doc-system                                         │
│  ├─ 创建 KIMI.md（AI操作手册）                               │
│  ├─ 创建 progress.txt（进度追踪）                            │
│  └─ 初始化项目文档结构                                       │
├─────────────────────────────────────────────────────────────┤
│  Step 5: workflow                                           │
│  └─ 按 plan.md 开始第一阶段开发                              │
├─────────────────────────────────────────────────────────────┤
│  Step 6: cost-control (总结)                                │
│  └─ 项目启动成本统计                                         │
└─────────────────────────────────────────────────────────────┘
```

## 输出文档

- `plan.md` - 项目开发计划
- `KIMI.md` - AI 操作手册（技术规范、约束）
- `progress.txt` - 进度追踪

## 使用示例

```
用户: /skill:new-project

AI: [执行 cost-control → socratic-inquiry]
    📊 成本基线: $0.005
    
    [socratic-inquiry] 需求澄清...
    🤔 让我们明确项目目标：
    1. 这个项目的核心目标是什么？
    2. 目标用户是谁？
    3. 主要功能模块有哪些？
    ...

用户: 回答...

AI: [执行 task-planner]
    📋 项目架构计划
    技术栈: Next.js + Prisma + TypeScript
    模块拆分: ...
    保存到 plan.md?

用户: 确认

AI: [执行 doc-system → workflow → /cost]
    📝 已创建 KIMI.md, progress.txt
    🚀 开始 Phase 1: 数据库设计
    📊 启动成本: $0.045
```
