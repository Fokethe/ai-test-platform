# AI Test Platform - UI布局优化需求文档

> 文档编号: ATP-PRD-004
> 版本: v1.0
> 创建日期: 2026-02-17

---

## 1. 问题描述

当前布局存在以下问题：
1. 底部功能区（快捷键、主题设置）占用侧边栏空间
2. 头像菜单中有重复的主题设置选项
3. Header区域过于空旷，可以整合功能

---

## 2. 优化目标

1. 将底部功能区移至右上角Header
2. 统一主题设置入口
3. 优化侧边栏空间利用
4. 提升整体视觉层次

---

## 3. 具体改动

### 3.1 当前布局
```
┌─ Sidebar ──────────┬─ Main Content ─────┐
│ 菜单项              │                     │
│ ...                 │                     │
│ 快捷键              │                     │
│ 主题                │    内容区域          │
│ [头像菜单]          │                     │
│   - 个人设置        │                     │
│   - 系统设置        │                     │
│   - 主题设置 ← 重复 │                     │
│   - 退出登录        │                     │
└────────────────────┴─────────────────────┘
```

### 3.2 优化后布局
```
┌─ Sidebar ──────────┬─ Header ───────────┬─ Main Content ─┐
│ 菜单项              │  🔍  🔔  🌙  ⌨️  [头像] │                │
│ ...                 │                     │    内容区域     │
│ [精简头像]          │  搜索 通知 主题 快捷键 │                │
└────────────────────┴─────────────────────┴────────────────┘

头像下拉菜单简化：
- 个人设置
- 系统设置
- 退出登录
（移除：快捷键、主题设置）
```

---

## 4. 组件调整

### Header新增元素
| 图标 | 功能 | 说明 |
|------|------|------|
| 🔍 | 全局搜索 | 快速搜索用例/套件 |
| 🔔 | 通知中心 | 系统通知入口 |
| 🌙/☀️ | 主题切换 | 浅色/深色切换 |
| ⌨️ | 快捷键 | 快捷键帮助 |

### 侧边栏简化
- 移除底部功能区
- 保留头像（仅显示）
- 点击头像展开简化菜单
