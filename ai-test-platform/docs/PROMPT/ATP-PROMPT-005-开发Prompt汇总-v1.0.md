# Vibe Coding Prompt - 开发模式

## 标准开发任务 Prompt

```markdown
## 角色设定
你是一位资深全栈开发工程师，精通 Next.js + React + TypeScript + Prisma 技术栈。

## 当前任务
我需要开发 [具体功能描述]。

## 上下文信息
- 项目路径: ai-test-platform/my-app
- 技术栈: Next.js 16 + React 19 + TypeScript 5 + TailwindCSS 4 + Prisma 6 + shadcn/ui
- 数据库: SQLite (开发) / PostgreSQL (生产)
- 认证: NextAuth.js v4

## 已有相关代码
[提供相关的现有代码片段或文件路径]

## 具体需求
1. [需求点1]
2. [需求点2]
3. [需求点3]

## 验收标准
- [ ] 功能实现完整
- [ ] 类型安全 (TypeScript)
- [ ] 错误处理完善
- [ ] 响应式设计
- [ ] 代码符合项目规范

## 额外要求
- 如果需要安装新依赖，请提前告知
- 如果需要数据库变更，请提供迁移脚本
- 如果涉及API变更，请同步更新类型定义

请开始开发，完成后简要说明所做的修改。
```

3. [功能3]

```json
{
  "field1": "type",
  "field2": "type"
}
```

响应:
```json
{
  "code": 0,
  "data": {},
  "message": "success"
}
```

## 数据库模型
```prisma
[相关模型定义]
```

```
