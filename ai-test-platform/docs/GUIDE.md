## 1. 快速开始

### 1.1 环境要求

- Node.js 18+
- npm 或 yarn
- Git

### 1.2 启动步骤

```powershell
# 1. 进入项目目录
cd F:\ai-test-platform\ai-test-platform\ai-test-platform\my-app

# 2. 安装依赖（首次）
npm install

# 3. 初始化数据库（首次）
npm run db:migrate
npm run db:seed

# 4. 启动开发服务器
npm run dev

# 5. 访问
http://localhost:3000
```

## 3. 常见问题排查

### 3.1 端口占用

```powershell
# 检查 3000 端口是否被占用
netstat -ano | findstr :3000

# 结束占用进程（将 <PID> 替换为实际进程ID）
taskkill /PID <PID> /F
```

### 3.2 页面空白或报错

```powershell
# 清除缓存重启
Remove-Item -Recurse -Force .next
npm run dev
```

### 3.3 数据库错误

```powershell
# 重新初始化数据库
npm run db:migrate
npm run db:seed
```

### 3.4 依赖缺失

```powershell
npm install
```

### 3.5 Compiling 卡住

```powershell
# 清除 Next.js 缓存
Remove-Item -Recurse -Force .next
npm run dev
```
