## 常见问题排查

### 问题 1: 页面空白或报错
```powershell
# 清除缓存重启
Remove-Item -Recurse -Force .next
npm run dev
```

### 问题 2: 数据库错误
```powershell
# 重新初始化数据库
npm run db:migrate
npm run db:seed
```

### 问题 3: 依赖缺失
```powershell
npm install
```
