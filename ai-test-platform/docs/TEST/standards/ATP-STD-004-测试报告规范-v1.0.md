## 3. 质量评估标准

### 3.1 通过标准
- 所有P0问题已解决
- P1问题解决率 >= 95%
- 测试用例执行率 = 100%
- 测试通过率 >= 90%

### 3.2 有条件通过标准
- 所有P0问题已解决
- 剩余P1问题有规避方案
- 测试用例执行率 >= 95%
- 测试通过率 >= 85%

### 3.3 不通过标准
- 存在未解决的P0问题
- 或P1问题解决率 < 80%
- 或测试通过率 < 80%
